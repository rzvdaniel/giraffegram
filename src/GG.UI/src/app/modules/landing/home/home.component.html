What is Giraffegram

Giraffegram is a software as a service which allows developers to manage transactional emails, and send them to their users from any programming language.

How it works

  1. Create an Account
  2. Login to the App
  3. Create an API key
  4. Add an Email template
  5. Send the email from your project (see Docs )

Open source

Giraffegram is 100% open source. You can host it yourself always under your control, or we can run it for you for a full serverless experience.
