What is Giraffegram

Giraffegram is a platform to manage and send transactional emails. Install it on premise for a one-time fee, or use our instance for a monthly subscription.

How it works

  1. Create email templates to use in your projects
  2. Send transactional emails as HTTP requests 
curl 
--location 'https://localhost:7160/api/email/send' \
--header 'Content-Type: application/json' \
--header 'X-Api-Key: 9yIQbmgbCdhpNDPcc3wutN2hH7cK==' \
--data-raw '{{'{'}}
    "from": {{'{'}}
        "email": "noreply@yourdomain.com",
        "name": "Your App Name"
    {{'}'}},
    "to": {{'{'}}
        "email": "tomjones@email.com",
        "name": "Tom Jones"
    {{'}'}},
    "template": "Welcome!",
    "variables": {{'{'}}
        "fullname": "Tom J. Jones"
    {{'}'}},
    "server": {{'{'}}
        "host": "youremailserver.com",
        "port": 465,
        "useSsl": true
    {{'}'}},
    "account": {{'{'}}
        "userName": "noreply@yourdomain.com",
        "userPassword": "*********"
    {{'}'}}
{{'}'}}

When would I use it?

When you build an app and need to send emails to your users. Giraffegram can help if you want to:

  • Manage email templates for your projects.
  • Send emails using simple HTTP requests.
  • Own the software, and install it on-premise.
  • Get the code, and change it to fit your needs.
